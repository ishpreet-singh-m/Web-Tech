🧳 Safar Saathi

Safar Saathi is a travel companion app that helps users capture, organize, and manage trip-related information.
It comes with a Flutter mobile app (frontend) and a Node.js + Express + MongoDB backend for storing and syncing data.

🚀 Features

📍 Trip Management – create, update, and view trips.

🗺️ Places to Visit – record and manage destinations.

🛠️ User Authentication – secure login/signup using JWT.

🔗 REST API – backend powered by Express + MongoDB.

📱 Cross-platform – built with Flutter, works on Android & iOS.

🏗️ Tech Stack

Frontend: Flutter (Dart)

Backend: Node.js, Express.js

Database: MongoDB

Authentication: JWT (JSON Web Tokens)

State Management (Frontend): Flutter Provider / Riverpod (depending on your setup)
